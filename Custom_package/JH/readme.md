# Custom_ROS_Package

## Description : Create ROS custom package to control drone
- [x]  SpaceArming 
- [x]  Takeoff + altitude select
- [x]  Set local postition(x,y,z) to move
- [x]  Return(RTL)
- [X]  Rotate 
- [X]  Brake 
- [X]  Move control (x+, x-, y+, y-)
- [X]  Exit Program
- [X]  Get current position

## Required :
- ROS
- Mavros(+Mavlink)
- Gazebo    
- Ardupilot(+Ardupilot_gazebo plugin)
